How can I join this very lovely SMP?

Here will be a short guide on how to join Legacy SMP. The server is playable on both Java and Bedrock Minecraft.

Joining on Java Edition (1.21.9-10)

The IP Address:
legacy-smp.uk
-----------------------------------------------------

Joining on Bedrock Edition

The Bedrock IP Address:
legacy-smp.uk
The Bedrock Port:
25683
-----------------------------------------------------

Joining on Bedrock Edition (Console)

If you play on Xbox, PlayStation or Switch / Switch 2 then you'll need to download the bedrock together app to join the server. The app can be found on the Apple App-store & Google Play-store.
