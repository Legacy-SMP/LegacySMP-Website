Welcome to Legacy SMP!

In this Minecraft Server, we try to turn Modern Minecraft into a refreshing wayback to the old console edition days like we used to have on our Xbox. Playstation, Nintendo Switch/Wii U and more! The SMP through it's time will expand and try to recreate the popular edition of the game, unfortunatly due to limitations we can't fully change the whole game to be like Minecraft Legacy Edition but with new tools added through each update of the game the more we are able to recreate the game! :3


The features we have on the Legacy SMP are:

Old Textures!

Play around in the classic textures in of the game!

Old Combat!

Fight mobs, players, etc with old-styled pvp!

Old Terrain Generation

Revisit the old terrain geneartion before the 1.18 Update!

Bedrock Support

With Geyser we're able to allow bedrock players to join the server with java players!

Friendly Community

When you join the server we do not tolerate bullying and harrasment so you can just fit right in! :3
